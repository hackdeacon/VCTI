## 二、战术常用黑话（中英对照）

### 1）进攻战术
- **Rush A/B/C**：快冲A/B/C点（抱团提速）
- **提速 / 提**：Push / Speed up（不慢摸，直接进）
- **慢摸 / 慢打**：Slow play / Creep（静步、控信息）
- **默认 / 控图**：Default（控中路、拿信息）
- **夹A/夹B**：Split / Flank（两路夹击包点）
- **转点**：Rotate（打不下换另一个点）
- **假打**：Fake（假装打A，实际打B）
- **前压**：Push forward（进攻方主动压防守前点）
- **控中**：Control Mid（拿中路控制权）
- **断后**：Cut off / Flank guard（防敌人绕后）
- **首杀 / 破点**：Entry frag / Opening frag（第一个进点杀人）

### 2）防守战术
- **架枪**：Hold / Angle（卡点等敌人）
- **卡点**：Hold angle
- **回防**：Retake（包点丢了重新打回来）
- **拖时间**：Delay / Stall
- **放包点 / 放守**：Give site（先放掉，等回防）
- **前顶**：Aggressive hold（防守方前压拿信息）
- **双架**：Double angle（两人交叉火力）

### 3）经济与局况
- **Eco局 / 经济局**：Eco（不买枪、省钱）
- **半起 / 强起**：Half buy / Force buy（经济差但买枪甲）
- **全起 / 满配**：Full buy（满甲+长枪+全技能）
- **保枪**：Save（不打了，保武器下回合用）
- **发枪**：Drop（给队友买枪）
- **连败奖励**：Loss bonus（连输加钱）

### 4）残局与个人能力
- **残局**：Clutch（1vN/少打多赢）
- **1v1 / 单挑**：1v1 / Duel
- **1v2 / 1v3**：1v2 / 1v3
- **翻盘**：Comeback
- **ACE**：ACE（一人杀完对方全队）
- **Team ACE**：Team ACE（每人各杀一个）
- **五杀**：Penta kill
- **carry / 带飞**：Carry
- **稳 / 保守**：Safe / Play safe
- **浪 / 激进**：Aggressive / Greedy
